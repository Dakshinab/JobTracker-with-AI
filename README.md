# JobTracker AI — Mobile App

A full-stack AI-powered job application tracker built with React Native, Expo, Supabase, and Groq AI. Available on both iOS and Android.

## Features

- **AI Job Analysis** — Paste any job description and get a match score, strengths, skill gaps, interview tips, and personalized career advice powered by Groq AI (LLaMA 3.3 70B)
- **AI Chance Analyzer** — When adding a job, AI instantly estimates your interview probability
- **Real-time Job Tracker** — Track all job applications with status (Applied, Interview, Offer, Rejected)
- **Smart Profile** — Add skills from 200+ IT skills database, work experience with duration auto-conversion
- **Authentication** — Full auth system with signup, login, forgot password powered by Supabase
- **Dark/Light Theme** — Automatic theme based on system preference
- **Cross-platform** — Runs on both iOS and Android from a single codebase
